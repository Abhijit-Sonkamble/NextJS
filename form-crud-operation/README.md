# 🚀 Next.js CRUD Application

A clean and scalable **CRUD (Create, Read, Update, Delete)** application built using **Next.js**. This project demonstrates full-stack capabilities with modern best practices, making it a solid foundation for real-world applications.

---

## 📌 Features

* ✅ Create new records
* 📖 Read and display data
* ✏️ Update existing entries
* ❌ Delete records
* ⚡ Fast and optimized with Next.js
* 📁 Organized folder structure (App Router)
* 🎯 Clean and maintainable code

---

## 🛠️ Tech Stack

* **Frontend:** Next.js (App Router)
* **Backend:** Next.js API Routes / Server Actions
* **Styling:** CSS / Tailwind (if used)
* **Language:** TypeScript / JavaScript

---


## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/your-username/your-repo-name.git
```

Navigate to project folder:

```bash
cd your-repo-name
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open in browser:

```
http://localhost:3000
```
